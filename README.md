# CloudPurr — Releases

Download the latest **CloudPurr.zip** from the
[Downloads release](https://github.com/butkevicius/CloudPurr-releases/releases/tag/downloads).

## First launch

CloudPurr is signed ad-hoc (not notarized), so macOS blocks the first launch:

1. Double-click **CloudPurr.app** (macOS refuses).
2. **System Settings → Privacy & Security → Open Anyway**, then confirm.

You only do this once — future updates install automatically via Sparkle.

`appcast.xml` in this repo is the Sparkle update feed; it is regenerated by the
maintainer's release script.

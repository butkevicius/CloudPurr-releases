# CloudPurr — Releases

Download the latest **CloudPurr.zip** from the
[Downloads release](https://github.com/butkevicius/CloudPurr-releases/releases/tag/downloads).

## Installation

1. Unzip **CloudPurr.zip** and drag **CloudPurr.app** into your **Applications** folder.
2. Double-click **CloudPurr.app** to open it.
3. If macOS blocks it, go to **System Settings → Privacy & Security → Open Anyway**, then confirm.

You only do this once — future updates install automatically.
